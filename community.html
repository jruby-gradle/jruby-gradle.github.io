JRuby/Gradle Community

Chat

The JRuby/Gradle community can be found primarily on the jruby-gradle-plugin Gitter chat hosted on gitter.im.

You can join the chat with your existing GitHub account.

Mailing List

For more asynchronous communications, the jruby-gradle mailing list can be used. If you wish to subscribe to the mailing list, email jruby-gradle+subscribe@googlegroups.com
